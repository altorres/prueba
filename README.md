## Install

#### descargar los paquetes de composer
```
composer install
```

#### [Bower](http://bower.io/search/?q=toastr)
```
bower install toastr
```

#### [npm](https://www.npmjs.com/package/toastr)
```
npm install --save toastr
```

#### [yarn](https://yarnpkg.com/en/package/toastr)
```
yarn add toastr
```

#### [Ruby on Rails](https://github.com/tylergannon/toastr-rails)
```ruby
# Gemfile

gem 'toastr-rails'
```

```coffee
# application.coffee

#= require toastr
```

```scss
// application.scss

@import "toastr";
```
